Pour lancer le programme, exécuter la classe Main.java qui se trouve dans le dossier Projet_algo_2019.

Pour comprendre la hiérarchisation du code, consulter le diagramme UML qui se trouve dans le fichier Compte_rendu.pdf du même dossier.

Au lancement du programme, apparition d'une fenêtre graphique représentant la ville avec des bâtiments jusque là construits.

Pour entrer une nouvelle dépense, cliquer sur le bouton "Nouvelle Entrée", remplir toutes les cases nécessaires puis cliquer sur OK. Cela conduit à l'obtention d'un nouveau bâtiment.

Pour voir les prévisions, cliquer sur "Statistiques". Des courbes apparaissent.

Pour fermer le programme, cliquer sur le bouton "Fermer". Cliquer sur la croix en haut à droite de la fenêtre ne permet pas de fermer le programme.
